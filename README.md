# Poll App using Livewire

A dynamic poll management application built with **Livewire**, enabling real-time interactions without writing custom JavaScript.

Powered by:

* Laravel Livewire
* Alpine.js

---

## Overview

This application allows to:

* Create and manage polls
* Add or remove poll options dynamically
* Vote on opinions
* Track votes in real time
* Experience instant validation without page reloads

---

## Features

* Real-time form validation
* Dynamic add/remove poll options
* Live vote tracking
* Seamless UI updates without page refresh
* Clean and maintainable architecture
* Built using Laravel + Livewire

---

## Tech Stack

* PHP
* Laravel
* Laravel Livewire
* Alpine.js
* Blade templating
* MySQL

## How It Works

### Poll Creation

* Users create a poll
* Options can be dynamically added or removed
* Real-time validation ensures clean input before submission

### Voting

* Users vote without page reload
* Votes update instantly
* Livewire handles state management on the backend

---

## Project Goals

* Demonstrate real-time validation using Livewire
* Showcase dynamic UI updates powered by PHP
* Provide a clean example for developers learning Livewire
